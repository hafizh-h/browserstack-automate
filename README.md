1. Buka PyCharm, switch ke branch okta-dev dengan perintah 'git checkout okta-dev' di terminal
2. Buka aplikasi XAMPP
3. Klik Start di Apache dan MySQL di aplikasi XAMPP
4. Buka localhost/phpmyadmin di browser
5. Klik menu Import di phpMyAdmin
6. Import file database db_automation_test.sql dari folder browserstack-automate
7. Run app.py di PyCharm
